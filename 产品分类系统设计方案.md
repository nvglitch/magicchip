# MagicChip 产品分类系统设计方案

## 一、设计目标

将现有产品页面从"点击分类直接跳转阿里国际站"改为：
1. **产品分类总览页** — 展示四大分类卡片，点击进入内部分类页
2. **分类产品列表页** — 展示该分类下的具体产品（如 MCAI35 等）
3. **产品详情页** — 展示单个产品的完整规格、图片、参数等

---

## 二、页面结构与路由设计

```
/products                    → 产品分类总览页（改造现有页面）
/products/[category]         → 分类产品列表页（新增）
/products/[category]/[id]    → 产品详情页（新增）
```

### 2.1 分类 ID 映射

| 分类 | category ID | 对应现有 JSON |
|------|-------------|---------------|
| Industrial Mini PC | `industrial-mini-pc` | `industrial-mini-pc.json` |
| Firewall Mini PC | `firewall-mini-pc` | `firewall-mini-pc.json` |
| Desktop Mini PC | `desktop-mini-pc` | `desktop-mini-pc.json` |
| 1U/2U Firewall Server PC | `firewall-server` | `firewall-server.json` |

## 七、实现步骤

1. **数据准备**
   - 创建 `content/products/items/` 目录
   - 为每个分类创建产品列表 JSON
   - 为每个具体产品创建详情 JSON

2. **页面开发**
   - 改造 `/products` 页面：修改链接为内部路由
   - 新建 `/products/[category]/page.tsx`：分类产品列表
   - 新建 `/products/[category]/[id]/page.tsx`：产品详情页

3. **组件开发**
   - 新建 `Breadcrumb.tsx` 面包屑组件
   - 新建 `ProductCard.tsx` 产品卡片组件
   - 新建 `SpecTable.tsx` 规格表格组件

4. **多语言**
   - 在 5 个语言文件中添加新文案

5. **测试**
   - 验证所有路由正常
   - 验证多语言切换
   - 验证响应式布局

## 九、示例效果预览
